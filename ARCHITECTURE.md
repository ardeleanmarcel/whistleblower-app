## Technology Justification

- **Next.js 16**: Full-stack framework with App Router for modern React patterns
- **TypeScript**: Type safety reduces bugs and improves developer experience
- **Prisma**: Type-safe ORM with excellent migration tooling
- **SQLite**: Zero-configuration database for development (easily upgradeable to PostgreSQL)
- **JWT**: Stateless authentication suitable for API-first architecture
- **Tailwind CSS**: Utility-first CSS for rapid, consistent UI development
- **bcryptjs**: Industry-standard password hashing with salt
- **nodemailer**: Reliable SMTP email delivery with HTML template support
